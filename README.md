# 🧑‍🤝‍🧑 KeenKeeper

## 📌 Project Description

KeenKeeper is a modern friendship management web application that helps users track, maintain, and nurture meaningful relationships. It allows users to monitor interactions like calls, texts, and video chats, ensuring no important connection is forgotten.

---

## 🚀 Technologies Used

* ⚛️ React.js
* 🎨 Tailwind CSS
* 📊 Recharts
* 🔄 React Router DOM
* 🔔 React Toastify
* 🧠 Context API

---

## ✨ Key Features

### 1. 📇 Friend Management

* View and manage a list of friends
* Detailed profile view for each friend
* Track last interaction and relationship status

### 2. 📅 Timeline Tracking

* Log interactions (Call, Text, Video)
* View activity history in a clean timeline
* Filter interactions by type

### 3. 📊 Analytics Dashboard

* Visual chart of interaction types
* Real-time statistics (Call/Text/Video)
* Clean and responsive donut chart UI
